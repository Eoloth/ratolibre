# Ratolibre.cl 🧭

**Ratolibre** es un sitio web hecho en React que muestra eventos culturales, deportivos y musicales en Chile. El objetivo es entregar información accesible y actualizada sobre panoramas, incluyendo referencias cruzadas y colaboradores destacados.

## Tecnologías utilizadas
- React (Vite)
- Tailwind CSS
- React Router DOM
- Carga de datos desde JSON
- Paginación
- Componentes reutilizables

## Cómo ejecutarlo localmente

```bash
npm install
npm run dev
```
🔑 Funcionalidades destacadas
Visualización de eventos paginados (6 por página)

Clasificación por etiquetas (deportivo, cultural, musical, etc.)

Perfiles de colaboradores

Carga dinámica de datos desde archivos JSON

Referencias cruzadas con otros sitios para mejorar visibilidad SEO
